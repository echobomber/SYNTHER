# synther
```
CSS 命名參考 BEM 架構
B (Block): 作為開頭，表示區塊。
E (Element): 使用雙下底線 __ 做分隔，表示為 B 區塊的元件。
M (Modifier): 使用雙 Dash -- 做分隔，表示為元件的狀態。
```
